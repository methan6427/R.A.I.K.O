R.A.I.K.O Agent
================

Quick Start:
1. Double-click "R.A.I.K.O Agent Setup" from the Start Menu
2. Enter your backend URL and auth token
3. Agent starts automatically

Manual:
- Edit config.json with your backend details
- Double-click raiko-agent.exe to run

Configuration:
- config.json: Backend URL, auth token, agent ID, etc.
- setup-gui.ps1: Interactive setup wizard
- setup.ps1: CLI setup wizard

For more info:
https://github.com/methan6427/R.A.I.K.O
